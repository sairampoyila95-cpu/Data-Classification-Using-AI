# Data Classification Using AI

## Project Overview

This project demonstrates a basic Machine Learning classification model using the Iris dataset. The model is trained using supervised learning techniques to classify flower species based on their features.

## Objective

* Load and understand a dataset
* Split data into training and testing sets
* Train a classification model
* Evaluate model performance
* Predict class labels for new data

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Dataset

The project uses the Iris Dataset, which contains:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Setosa
* Versicolor
* Virginica

## Machine Learning Workflow

1. Load Dataset
2. Preprocess Data
3. Split Data into Training and Testing Sets
4. Train Decision Tree Classifier
5. Make Predictions
6. Evaluate Accuracy
7. Visualize Results

## Output

* Dataset information
* Training and testing data statistics
* Classification report
* Accuracy score
* Visualization graph
